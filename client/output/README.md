# Output Directory

This directory contains the aggregated telemetry data output files generated by the client application.

Each run creates a new JSON file with the timestamp: `telemetry_[timestamp].json`

## File Format

```json
{
  "success": true,
  "totalDevices": 500,
  "devices": [
    {
      "serialNumber": "SN-000",
      "timestamp": "2026-02-03T18:06:32.000Z",
      "power": 4523,
      "voltage": 235,
      "current": "18.45",
      "temperature": "42.3",
      "efficiency": "98.76",
      "status": "online"
    }
  ],
  "errors": [],
  "statistics": {
    "totalRequests": 50,
    "successfulRequests": 50,
    "failedRequests": 0,
    "retriedRequests": 0,
    "totalDevices": 500,
    "totalBatches": 50,
    "duration": 50234,
    "averageRequestTime": 1004.68
  }
}
```
